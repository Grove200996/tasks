## Git commands

*  **git init** - используется для указания папки, в которой будет создан репозиторий и будут отслеживаться изменения.
* **git add** - с помощью этой комманды мы добавляем содержимое файла для последующего коммита.
* **git commit** - создаём коммит (фиксируем и сохраняем)

* **git status** - Показывает текущее состояние гита, есть
ли изменения, которые нужно закоммитить
(сохранить)

* **git log** - Журнал изменений

* **git checkout** - переключение между версиями (git checkout master для возвращения в рабочую ветку)

* **git diff** - Показывает разницу между текущим файлом
и сохранённым

* **git branch** - Показать текущую ветку

* **git branch -d <branch_name>** - удаление ветки

* **git remote -v** - Список ваших удаленных подключений к другим репозиториям.

* **git remote rm <name>** - git remote rm <name>

* **git add remote <repository_name>** - Создание нового подключения к удаленному репозиторию.

* **git merge <branch_name>** - соединить ветки

* **git push** - загрузить изменения на удаленный репозиторий

* **git pull** - используется для загрузки содержимого из указанного удаленного репозитория

* **git checkout -b <branch_name>** - создать новую ветку и сразу перейти в неё


*Чтобы вызвать ранее введённую команду,
пользуемся стрелками на клавиатуре.
Перебираем недавно введённые команды
нажатием стрелки «вверх»*

**echo file > file's name** - создать файл через терминал, где *file* - содержимое файла.
